# tool-os-win-wsl

## guide

- 是否有必要从ubuntu切换到wsl
  - 是自己笔记本电脑的问题
    - 不管是在win下还是ubuntu下，都会出现找不到硬盘的情况
    - 都会无法开机

- wsl优点
  - 同时使用优秀的win工具和linux工具，如office excel、shell

- wsl缺点
  - 暂时不完善，特别是对GPU的支持
  - 跨系统访问文件和文件夹效率不高，文件路径不简单
  - 被 docker 的网络和 wsl2 的网络问题搞了一上午。妈的。还没解决。
