# toolchain
- [Text Embeddings Inference](https://huggingface.co/docs/text-embeddings-inference/index)
  - Text Embeddings Inference (TEI) is a comprehensive toolkit designed for efficient deployment and serving of open source text embeddings models. 
  - It enables high-performance extraction for the most popular models, including FlagEmbedding, Ember, GTE, and E5.
  - Optimized Inference: TEI leverages Flash Attention, Candle, and cuBLASLt by using optimized transformers code for inference.
  - Safetensors weight loading: TEI loads Safetensors weights for faster boot times.
