# keyboardnotes
- features
  - keyboard first
  - no vendor lock-in
    - all in markdown text format, with opt-in extended features
  - fast with modern web tech stacks
  - beautiful design
  - cloud sync, secure by auth0

- xp
  - 首页默认左右双栏布局(文档列表/预览)，通过jk按键可快速切换预览
  - 按enter可进入编辑模式，默认左中右3栏布局(文档列表/编辑/预览)
