# pm-documentation-ppt

# features

- core/核心功能
  - 读取markdown内容生成DOM，然后渲染

- block style editor
- 对于包含数据(表格)的文档，实现类似cboard的数据联动
  - 文档block实现为展示块和控制块
  - 在控制块更新数据，所有展示块的内容都会更新

- 标题元数据提取
  - 时间、地点、人物
