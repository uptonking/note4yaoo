# note-ux-style-pattern

## interaction

- 文字逐个消失，再逐个出现，可以突出重点

- hightlight items on page and block user interactions
  - https://kamranahmed.info/driver.js/
- unfold: 揭开效果，类似画皮的撕皮

- 骨架屏效果
