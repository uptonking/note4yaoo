# work-xp
- [三年前端面试经验加感悟](https://juejin.cn/post/6844904089902383112)
  - 我经历的面试，很多情况下就是让你给出解决方案，考得就是技术的实际运用能力，
    - 几乎每一轮面试我都会碰到这种题目，这就要求你在平时工作中要有意识的积累，光搬砖你是说不出来完整的解决方案的
  - 不会涉及到具体的问题点，考察的就是候选人的知识广度和深度，平时的积累，考虑问题的维度等方面
    - 如果我是候选人，我可能会从长列表的优化、跨域/跨页面通信、配置化（可维护性）、数据预加载、数据缓存、页面/组件无缝切换/转场动画、离线包/ssr等角度来回答这个问题
  - 技术发展快，这条路不进则退，只有持续学习才能在技术这条路上越走越远
  - 能够把自己的技能展现出来也是一种本事，在面试的时候尤为重要，头疼
