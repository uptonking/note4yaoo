# pick-dict-common-prefix-suffix

# suffix

- -esque/-like
  - 似的，类... 风格的
- -itis
  - 过度的，沉迷于；…病
  - televisionitis 电视迷

# prefix

- dys-
  - 不正常的
  - 坏的，困难的

- octo-
  - 八；八…的(构成名词、形容词和副词)
