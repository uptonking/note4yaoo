# extensions
- https://github.com/CarterLi/echarts-with-offscreencanvas
  - Echarts with OffscreenCanvas
  - A canvas cannot be used directly in worker, but OffscreenCanvas can be used as a handle to control it.
  - All rendering is done in worker.
  - The `OffscreenCanvas` interface provides a canvas that can be rendered off screen, decoupling the DOM and the Canvas API so that the `<canvas>` element is no longer entirely dependent on the DOM.
