# tool-software-usage

## 软件使用体验xp
- Ubuntu开启显示隐藏文件后，文件夹的大小会变大，如git项目下的.git目录有时非常大
- 常用端口(默认可不写)
  - HTTP：80
  - HTTPS：443
  - SSH：22
  - DNS：53
  - FTP：21
  - MYSQL：3306
- 可用ping检测ip，用telnet检测端口
