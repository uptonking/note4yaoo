# pm-list-grid-table-ux

## faq-not-yet

- 对于list或grid中的项目，拖动A到B的位置，实现的效果可以有2种
  1. A到B的位置，B到A的位置，其他不变
  2. A到B的位置，然后A之后B之前的所有元素前移一位，最后B挨着A
  - 从性能角度考虑，选第一种
