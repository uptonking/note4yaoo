# wiki-iot-keyboard-symbols

```
` -- Back quote
~ -- Tilde

! -- Exclamation point (a.k.a. BANG!)
@ -- At sign
# -- Hash sign
$ -- Dollar
% -- Percent
^ -- Caret
& -- Ampersand
* -- Star /asterisk / splat
( -- Open Parenthesis
) -- Closed Parenthesis

- -- Dash
_ -- Underscore
= -- Equals
+ -- Plus


[ -- Open square bracket
] -- Closing square bracket
{ -- Open curly bracket
} -- Closing curly bracket
\ -- Back slash
| -- Pipe

; -- Semi-colon
: -- Colon
' -- Single quote
" -- Double quote

, -- Comma
< -- Less than
. -- Dot / period
> -- Greater than
/ -- Slash
? -- Question mark
```
