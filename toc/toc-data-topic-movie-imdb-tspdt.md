# imdb-data
- https://github.com/nas5w/imdb-data
  - A JSON file of 50, 000 IMDB movie reviews to be used in machine learning applications.
  - "Large Movie Review Dataset" from Stanford University in json format

- https://github.com/vegidio/js-vimdb
  - This package lets you get IMDb information from movies, series, etc without an API key.

- https://github.com/worr/node-imdb-api
  - A non-scraping, functional node.js interface to imdb
  - an api key is required

- https://github.com/iiiiiii1/douban-imdb-api
  - 基于豆瓣、IMDB、烂番茄评分的电影/电视剧双语(中英)数据api接口

- https://github.com/imdeepmind/processed-imdb-wiki-dataset
  - IMDB WIKI dataset is the largest dataset of human faces with gender, name and age information
  - [IMDB-WIKI - 500k+ face images with age and gender labels](https://data.vision.ee.ethz.ch/cvl/rrothe/imdb-wiki/)

- https://github.com/transitive-bullshit/populate-movies
  - This project resolves a full set of movies from TMDB and IMDB in batches, storing them into a Postgres database using Prisma.
  - This includes difficult-to-access data like IMDB ratings, Rotten Tomatoes audience & critic scores, and YouTube trailers.
  - Everything is handled including data fetching, normalization, cleaning, and filtering.
- https://github.com/transitive-bullshit/next-movie
  - An example Next.js 13 app built using the new router, server components, and all the latest hotness
  - uses populate-movies to generate it's high quality movie database
