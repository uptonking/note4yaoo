# pm-office-siyuan-xp

# guide

- features
  - 本地优先，离线可用
  - 加密同步，隐私安全
    - 支持docker部署
  - 支持多设备
    - 支持浏览器扩展，但不支持网页版
    - mobile：android、ios
    - desktop：mac、linux
  - 融合块、大纲和双向链接
    - rich blocks
    - 块级双向链接
    - 块可拆分、重组
    - 列表可折叠
  - wysiwyg 块编辑器

- cons
  - 支持浏览器扩展，但不支持网页版直接使用
    - 在本地desktop应用可以打开浏览器 http://127.0.0.1:6806/stage/build/desktop 使用整个程序
  - 自动生成的本地文件名为uuid，而不是可读文件名
  - 图片
    - 不支持在线图片链接，只支持本地图片

- user-xp
